# oss-student-resources
This repository collects and points to ways that GT students can engage with open source projects.

- [GT Student Open Source Engagement](https://github.com/gt-ospo/oss-student-resources/blob/main/gt-student-oss-engagement.md)
    - A comprehensive overview of the programs and pathways available for Georgia Tech students to engage with open source software research, covering paid, for-credit, and community opportunities.
- [Tips for Mentoring Students with OSS](tips-for-mentors.md)
    - A practical guide for open source maintainers on how to make their projects accessible and welcoming to student contributors, from writing clear contributor guidelines to using GitHub templates and CI/CD.

This set of resources is maintained by [Georgia Tech's Open Source Program Office](ospo.cc.gatech.edu)
